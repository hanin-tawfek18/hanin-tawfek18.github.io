What is Kotlin?

Kotlin is a cross-platform, statically typed, general-purpose high-level programming language with type inference. Kotlin is designed to interoperate fully with Java, and the JVM version of Kotlin's standard library depends on the Java id Library, but type inference allows its syntax to be more concise.


Why to learn Kotlin?

  • Easy to learn
  • Interoperable with Java
  • Extensions
  • No more Null Pointer Exceptions
  • Reliable and safe
  • Strong support


Skills you will gain after the course:

  • Kotlin Playground
  • practice using and extending protocols.
  • declare and initialize different types of variables
  • create arrays
  • create control flow patterns using conditionals and loops
  • Principles of software development
  • Kotlin coroutines
